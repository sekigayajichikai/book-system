📋 システム概要

関ヶ谷自治会の会館（会議室・和室・図書室）の予約管理と、自治会カレンダーを一つにまとめたシステムです。

3つの使い方

利用者できること
一般の方カレンダー・会館予約状況の閲覧
登録団体閲覧 + 予約申請 + マイページ
管理者カレンダー管理・インポート・申請承認・団体管理・設定

👀 一般の方向け（閲覧）

ログインなしで、2つのタブを確認できます。

カレンダータブ

自治会の予定一覧が月ごとに表示されます。

  • ◀ ▶ で月を切り替え
  • 「主な予定」欄に重要なイベントがまとめて表示されます
  • 祝日は赤色、土曜は青色で表示
  • 休館日にはオレンジの「休館」バッジが付きます

会館予約状況タブ

会館の各部屋の予約状況が月間カレンダーで確認できます。

  • 日付セルに予約が色分けで表示されます
  • セルをクリック → その日の予約一覧がポップアップ表示
  • 予約をクリック → 詳細（団体名・時間・部屋）を確認
  • 左のサイドバーで表示する団体を絞り込めます
部屋の色は凡例で確認できます。会議室、和室、図書室

🔑 団体ログイン

登録済みの団体は、ログインすると予約機能が使えるようになります。

  1. 画面右上の 「ログイン」 ボタンをクリック
  2. ドロップダウンから 団体名 を選択
  3. 管理者から発行された パスコード を入力
  4. 「ログイン」ボタンを押す

ログインすると、ヘッダーに団体名が表示され、「予約」タブと「マイページ」タブが使えるようになります。

パスコードがわからない場合は、自治会事務局にお問い合わせください。

📝 予約の仕方

団体ログイン後、「予約」タブから会館の予約ができます。

空き状況の確認

  • 月表示：月間の全部屋の予約を一覧で確認
  • 週表示：1週間の時間帯別の空き状況を確認
  • 上部の部屋ボタンで特定の部屋に絞り込めます

予約の手順

  1. 予約したい日時の空きスロットをクリック
  2. 予約フォームが開きます。以下を入力してください：
    • 利用区分（料金に影響します）
    • 部屋・時間帯（クリックした内容が初期値）
    • 備品（必要な場合はチェック、各300円）
    • メモ（任意）
  3. 料金を確認して 「予約する」 ボタンを押す
  4. 管理者の承認後、予約が確定します

利用料金の目安

利用区分料金
自治会運営・各部の会議無料
会員の趣味・同好会・親睦100〜200円
混成団体・教室200〜500円
弔事の宿泊1,000円
その他個別設定
予約は申請後すぐには確定しません。管理者の承認を待ってください。マイページで状況を確認できます。

📱 スマホでの使い方

スマホでアクセスすると、自動的にスマホ向け画面に切り替わります。

タブの切り替え

画面上部の2つのタブで切り替えます：

  • カレンダー：月間の予定一覧（日ごとのカード形式）
  • 会館予約状況：月間の予約状況（日ごとのカード形式）

操作方法

  • ◀ ▶ ボタンで月を切り替え
  • 左右スワイプでも月を切り替えられます
  • 今日の日付まで自動スクロールします
ログイン後は「予約」「マイページ」タブも表示されます。

🛡 管理画面ログイン

  1. URLの末尾に #admin を付けてアクセス
    （例：https://sekigaya-calendar.vercel.app/#admin）
  2. 管理者パスワードを入力して「ログイン」

ログイン後、上部のタブから各管理機能にアクセスできます：

  • カレンダー ─ カレンダー・予約の管理
  • インポート ─ Excel/Googleスプレッドシートの取込
  • 申請 ─ 団体からの予約申請の承認/却下
  • 団体マスタ ─ 登録団体の管理
  • 設定 ─ 利用区分・備品・グループなどのマスタ管理

📅 管理：カレンダー

「カレンダー」タブでは、カレンダーと会館予約の2つのサブビューがあります。

予定サブビュー

自治会のイベント・予定を管理します。

  • 空きセルをクリック → 新しい予定を作成
  • 既存の予定をクリック → 詳細表示・編集・削除
  • 月/週/一覧の表示切替が可能

会館予約状況サブビュー

会館の予約を管理します。

  • 空きセルをクリック → 新しい予約を作成
  • +N件をクリック → その日の予約一覧を表示
  • 予約アイテムをクリック → 詳細の確認・編集
左のサイドバーで団体のフィルタリング、「主な予定」の表示/非表示を切り替えられます。

📥 管理：インポート

Excelファイルまたは Googleドライブから予約データを一括取込できます。

会館予約のインポート

方法1：Googleドライブから取込

  1. 「Googleドライブから取込」をクリック
  2. 自動的にスプレッドシートを読み取り、差分を計算します

方法2：Excelファイルをアップロード

  1. 「Excelファイルをアップロード」エリアにファイルをドラッグ&ドロップ（またはクリックして選択）
  2. ファイルが読み込まれ、差分が表示されます

差分の確認と承認

取込後、以下の差分タイプが表示されます：

ラベル意味
新規新しく追加される予約
変更タイトル等が変更された予約
差異既存とタイトルが異なる予約（手動確認用）
削除Excelから消えた予約

団体の紐付け

各行の右側にグループ → 団体の2段ドロップダウンがあります。

  • タイトルからの自動マッチが行われます（例：「図書部定例会」→「図書部」）
  • 自動マッチされなかった場合は手動で選択してください
  • まずグループを選び、次に団体名を選択します

反映の手順

  1. 差分を確認し、各行の ✓（承認）または ✕（却下）を選択
  2. 「全て承認」ボタンで一括承認もできます
  3. 必要に応じて団体を選択・変更
  4. 更新日を入力し、「反映する」 ボタンを押す
反映すると予約データが上書きされます。差分をよく確認してから反映してください。

一般予定・休館日のインポート

「予定スプレッドシートから取込」で、一般予定と休館日をGoogleスプレッドシートから取り込めます。

✅ 管理：申請承認

団体からの予約申請を承認または却下します。

  1. 「申請」タブを開く
  2. ステータスフィルタで「承認待ち」を選択
  3. 申請内容（日時・部屋・団体名）を確認
  4. 承認 → 予約確定、却下 → 理由を入力して却下
却下理由は、申請した団体のマイページに表示されます。

👥 管理：団体マスタ

登録団体の追加・編集・管理を行います。

団体の登録情報

項目説明
団体名表示名（必須）
ふりがな検索・並び替え用
グループ所属グループ（自主活動部、委員会など）
パスコード団体ログイン用の暗証番号
代表者名代表者の氏名
連絡先メールアドレス
活動内容団体の活動概要
キーワードインポート時にタイトルから団体名を自動推測するためのキーワード

基本操作

  • 左のリストから団体を選択 → 右側に詳細フォームが表示
  • 「新規追加」で団体を追加
  • 編集後「保存」で更新
  • 「削除」で団体を削除（確認あり）

⚙ 管理：設定

システムのマスタデータを管理します。

管理できる項目

項目内容
利用区分予約フォームに表示される利用カテゴリと料金
備品貸出備品の種類と料金
団体グループ団体の分類グループ（自主活動部、委員会など）
施設利用可能な部屋の設定

各項目は追加・編集・削除・並べ替えが可能です。

設定の変更は即座に反映されます。料金やカテゴリの変更は慎重に行ってください。

📖 カレンダー管理の流れ

カレンダーの更新作業は、大きくA. 会館日程表の読込とB. その他の予定の記入の2ステップです。

A. 会館日程表の読込

ステップ1：エクセルの読込

  1. 管理画面の「インポート」タブを開く
  2. 「Googleドライブから取込」をクリック
    ※ ドライブから読込できない場合は、横山さんにメールでエクセルファイルをもらい「Excelファイルをアップロード」からアップロード

ステップ2：差分のチェック

読み込むと、差分の一覧が表示されます。以下の点を確認してください：

  • 団体名が正しく紐付いているか（グループ→団体のドロップダウンで確認・修正）
  • 日付や時間に間違いがないか
新規追加だけでなく、変更や削除の差分も提案されます。それぞれの内容をよく確認してから承認してください。

ステップ3：承認と反映

  1. 各行の内容を確認し、✓（承認）または ✕（却下）を選択
    「全て承認」で一括承認もできます
  2. エクセルが実際に更新された日を「更新日」欄に入力
  3. 「反映する」ボタンを押して完了

ステップ4：微調整（余力があれば）

読込が完了したら、必要に応じてカレンダー上で調整します：

  • 「会館予約状況」タブ → 一覧ビューで、タイトルを調整
    ※ このタイトル変更はカレンダービューでも可能
  • 「カレンダー」タブで、イベント時刻の入力、詳細を住民向けに編集

B. 会館日程表に記載のない予定の記入

ステップ1：予定の追加

エクセルに載っていない予定を手動で追加します。

  • 情報源：関ヶ谷だより、回覧板のチラシなど
  • 操作：「カレンダー」タブで、該当日のセルをクリック → 予定を作成

ステップ2：主な予定にマークをつける

住民に特に知らせたい重要なイベントには「主な予定」のチェックをつけます。

  • 予定の編集画面で ☆（主な予定）をONにする
  • マークをつけた予定は、カレンダー上部の「主な予定」欄にまとめて表示されます
作業の頻度の目安：毎週金曜日にA→Bの順で作業してください。
